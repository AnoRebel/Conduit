## Release v1.0.1

### Initial Release

### Installation

#### JSR
```bash
bunx jsr add @conduit/client@1.0.1
bunx jsr add @conduit/server@1.0.1
# or with npx
npx jsr add @conduit/client@1.0.1
npx jsr add @conduit/server@1.0.1
# or with deno
deno add jsr:@conduit/client@1.0.1
deno add jsr:@conduit/server@1.0.1
```

#### Go
```bash
go get github.com/AnoRebel/Conduit/packages/go-client@v1.0.1
```

### Docker

```bash
docker pull ghcr.io/anorebel/conduit/server:1.0.1
docker pull ghcr.io/anorebel/conduit/server-admin:1.0.1
docker pull ghcr.io/anorebel/conduit/admin-ui:1.0.1
```
